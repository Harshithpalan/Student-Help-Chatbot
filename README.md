# 🎓 CampusAI | Student Help Chatbot

A modern, premium, and responsive React-based chatbot interface designed to assist students with campus life, academic inquires, and administrative tasks.

---

## ✨ Features

- **💎 Premium Design**: High-end "Glassmorphism" interface with dynamic mesh gradients and a refined dark theme.
- **⚡ Smart AI Logic**: A structured local knowledge base providing instant answers for Library hours, Enrollment dates, and Scholarships.
- **🎬 Fluid Animations**: Smooth message transitions and AI typing indicators powered by `Framer Motion`.
- **📱 Fully Responsive**: Optimized for both high-resolution desktops and mobile devices.
- **🔍 Quick Actions**: Interactive sidebar for one-click access to common student queries.

---

## 🛠️ Technology Stack

- **Framework**: [React 19](https://react.dev/)
- **Build Tool**: [Vite 6](https://vitejs.dev/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Styling**: Vanilla CSS (Modern CSS Variables & Backdrop Filters)

## 🚀 Getting Started

### 1. Prerequisites
Ensure you have [Node.js](https://nodejs.org/) installed on your machine.

### 2. Installation
Clone the repository and install the dependencies:

```bash
npm install
```

### 3. Running the App
Start the development server:

```bash
npm run dev
```

The app will typically be available at `http://localhost:5173/`. If that port is in use, Vite will automatically try `5174` or `5175`.

---

## 🔧 Troubleshooting

### **Import Analysis Error**
If you encounter a `[plugin:vite:import-analysis]` error regarding missing dependencies on Windows, the project is pre-configured with explicit aliasing in `vite.config.js` to resolve this. Ensure you run:

```bash
npm install --force
```
